# Magnu Imperiu features

> Daily empire guessing game. How well do you know history?

Homepage: [https://magnu-imperiu.com](https://magnu-imperiu.com)

## Feature highlights

History is usually taught as dates and names, which is the worst way to remember it. Magnu Imperiu flips that: a free daily puzzle where you guess a mystery empire from geographic clues, dynasty hints, time period, and size. Six attempts per day, a new empire every 24 hours, 50+ empires in rotation. Teachers have started using it as classroom warm-ups; history buffs run daily streaks.

## In depth

Magnu Imperiu turns history into a daily ritual the way Wordle turned vocabulary into one. Every 24 hours a new mystery empire appears, and you get 6 attempts to identify it using geographic clues, time period, dynasty hints, and size comparisons. The library covers 50+ great empires from the Achaemenids and Han to the Mongol Khanate and the Ottomans. Teachers use it as classroom openers; history buffs use it as a daily streak.
